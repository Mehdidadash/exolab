# exolab
وب‌سایت لابراتوار دیجیتال اگزولب

## توضیحات اولیه
این پروژه شامل یک سایت PHP ساده به همراه پنل مدیریت برای مدیریت لیست قیمت و نمونه کار است.

## ویژگی‌ها
✅ سایت تماماً محلی (بدون CDN)
✅ لیست قیمت‌ها از پایگاه داده
✅ گالری نمونه کار با Lightbox
✅ شبکه‌های اجتماعی (اینستاگرام، بله، روبیکا)
✅ مسیریابی (نشان، بلد، گوگل مپ)
✅ پنل مدیریت ساده و امن
✅ مدیریت تصاویر با آپلود

## ساختار فایل‌ها
```
exolab/
├── index.php                    # صفحه اصلی
├── config.php                   # تنظیمات پایگاه داده
├── db.php                       # اتصال و توابع دیتابیس
├── contact.php                  # اطلاعات تماس و شبکه‌های اجتماعی
├── install.php                  # نصب اولیه
├── includes/
│   └── icons.php               # آیکون‌های SVG
├── assets/
│   ├── css/
│   │   ├── style.css           # استایل اصلی
│   │   └── lightbox.css        # استایل گالری
│   ├── js/
│   │   ├── main.js             # اسکریپت‌های عمومی
│   │   └── lightbox.js         # Lightbox ساده
│   ├── uploads/                # تصاویر آپلود شده (برای نمونه کار)
│   ├── fonts/
│   ├── lib/
│   └── fonts/
└── admin/
    ├── auth.php                # احراز هویت و تابع‌های مدیریت
    ├── login.php               # صفحه ورود
    ├── dashboard.php           # داشبورد
    ├── logout.php              # خروج
    ├── prices.php              # لیست قیمت‌ها
    ├── price_form.php          # فرم قیمت
    ├── save_price.php          # ذخیره قیمت
    ├── delete_price.php        # حذف قیمت
    ├── works.php               # لیست نمونه کار
    ├── work_form.php           # فرم نمونه کار
    ├── save_work.php           # ذخیره نمونه کار
    └── delete_work.php         # حذف نمونه کار
```

## نصب اولیه
1. اطلاعات دیتابیس خود را در `config.php` تنظیم کنید.
2. پوشه `assets/uploads` را ایجاد کنید (یا فایل `install.php` خودکار آن را می‌سازد).
3. `install.php` را در مرورگر اجرا کنید: `http://exolab.ir/install.php`
4. پس از اجرای موفق، فایل `install.php` را حذف کنید.
5. به `admin/login.php` بروید و وارد شوید.

## حساب مدیر اولیه

- رمز عبور: `admin123`

⚠️ **توصیه شدید**: پس از ورود، این رمز را تغییر دهید.

## راهنمای استفاده

### افزودن قیمت
1. در پنل مدیریت، به «قیمت» بروید.
2. دکمه «افزودن قیمت جدید» را کلیک کنید.
3. اطلاعات را پر کنید و ذخیره کنید.

### افزودن نمونه کار
1. در پنل مدیریت، به «نمونه کار» بروید.
2. دکمه «افزودن نمونه کار جدید» را کلیک کنید.
3. عنوان، توضیح و تصویر را انتخاب کنید.
4. ترتیب نمایش را تنظیم کنید.
5. ذخیره کنید.

## تکنولوژی‌های استفاده شده
- **PHP 7+**: سمت سرور
- **MySQL/MariaDB**: پایگاه داده
- **HTML5 + CSS3**: رابط کاربری
- **Vanilla JavaScript**: رفتار سایت
- **SVG**: آیکون‌ها
